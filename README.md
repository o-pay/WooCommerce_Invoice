# WooCommerce  歐付寶電子發票 
---

## 1. 介紹

  - 歐付寶電子發票外掛套件，提供合作特店以及個人會員使用開放原始碼商店系統時，無須自行處理複雜的檢核，直接透過安裝設定外掛套件，便可快速介接歐付寶系統，進行電子發票一般開立。

  - Wordpress 4.5.3 / 4.6.1 / 4.7.3 / 4.8.0~4.8.2
  - WooCommerce 2.6.0 ~ 2.6.14 / 3.1.0 ~ 3.1.2 / 3.2.0
